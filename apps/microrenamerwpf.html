Desktop App

MicroRenamerWPF

Desktop file-renaming and image-prep tool built for Syntax Communications. Has a "PWA Resize Tool" tab that embeds my web companion app (MicroRenamerWEB) right inside the desktop window via WebView2 — so the renamer and the resize/gallery tools live in one app.

C#WPF ImageSharpOpen XMLWebView2

Core MVP

  • Bulk file renaming & text cleanup
  • Case conversion, date stamping, bullet/title tools
  • Embedded "PWA Resize Tool" tab
  • Preset clipboard text snippets

Employer Skills

  • WPF desktop UI
  • WebView2 embedding of a web app
  • ImageSharp & Open XML integration
  • Multi-tool tabbed app design

Future Features

  • Additional renaming presets
  • More PWA tools surfaced in-app
  • Settings persistence across sessions
