# m1sh3ll.github.io

The static, GitHub Pages-hosted version of my personal portfolio site — served at [m1sh3ll3.com](https://m1sh3ll3.com).

## About

I'm Michelle Nesbitt, a C#/.NET developer with an e-commerce operations background. This site showcases the internal tools I've built on the job and on my own — desktop apps, web apps, and everything in between.

This repo is a static HTML/CSS/JS snapshot of the site, deployed via GitHub Pages. The primary, actively developed version is a Blazor Server app — see [m1sh3ll3.com](https://github.com/m1sh3ll/m1sh3ll3.com).

It also hosts **Project Unify**, a self-directed packaging framework where I present years of real, already-built tooling as one coherent portfolio. Every app shown on the site is real software, built for actual employers or personal projects.

## Tech Stack

- HTML / CSS / JavaScript
- GitHub Pages

## Project Structure

```
index.html      Main portfolio page
styles.css      Site styling
script.js       Page interactivity (slideshows, nav, etc.)
apps/           Individual project detail pages
images/         Project screenshots
```

## Running Locally

This is a static site — just open `index.html` in a browser, or serve the folder with any static file server.
